## Create a new repository by gitbash command line
* git init
* git status
* git add MyProject/
* git commit -m "My First Commit"
* git branch -M main
* git remote Origin URL
* git push -u Origin main
---
### Push Files to an existing repository
* git remote add Origin URL
* git branch -M main
* git push -u Origin main
